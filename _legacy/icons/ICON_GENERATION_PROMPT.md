# Prompt pour génération d'icônes - Grindr Auto Tap Extension

## Prompt principal
```
Crée une icône d'extension de navigateur représentant un outil d'automatisation pour une application de rencontre. 

L'icône doit montrer :
- Un doigt stylisé qui tape/appuie sur un écran ou un bouton (représentant l'action "tap" automatisée)
- Des éléments visuels suggérant l'automatisation : des engrenages subtils, des flèches circulaires, ou des lignes de code discrètes
- Un petit graphique ou des barres statistiques en arrière-plan (représentant les statistiques collectées)
- Une palette de couleurs principale : orange/rouge corail (#FF6B6B ou similaire) pour évoquer l'application cible, avec des accents blancs ou gris clair

Style :
- Design moderne, minimaliste et épuré
- Style flat design ou légèrement isométrique
- Lignes nettes et formes géométriques simples
- Haute lisibilité même à petite taille (16x16 pixels)
- Pas de texte, uniquement des symboles et formes
- Fond transparent ou solide avec bon contraste

Éléments de composition suggérés :
- Au centre : un doigt stylisé (vue de côté) qui pointe vers un cercle/bouton
- En arrière-plan : des engrenages ou des flèches circulaires discrètes (opacité 30-40%)
- En bas ou sur le côté : de petites barres de graphique ou un compteur numérique stylisé
- Tout doit être harmonieusement intégré dans un carré

Format : PNG avec fond transparent
Résolution : Haute résolution pour permettre le redimensionnement
Couleurs : Principalement orange/rouge (#FF6B6B, #FF5252) avec blanc (#FFFFFF) et gris (#F5F5F5)
```

## Variantes pour différentes tailles

### Version détaillée (128x128)
```
Même prompt que ci-dessus, mais avec plus de détails visibles :
- Les engrenages peuvent être plus visibles
- Le graphique peut avoir plus de barres
- Plus de nuances dans les couleurs
```

### Version moyenne (48x48)
```
Même prompt que ci-dessus, mais simplifié :
- Moins de détails dans les engrenages
- Graphique simplifié (3-4 barres maximum)
- Formes plus grandes et plus lisibles
```

### Version petite (16x16)
```
Version ultra-simplifiée pour la petite taille :
- Seulement le doigt stylisé et le bouton/cercle
- Pas d'engrenages (trop petits)
- Peut-être juste une petite barre de graphique en bas
- Formes très simples et contrastées
- Focus sur la lisibilité maximale
```

## Prompt alternatif (style plus technique)
```
Icône d'extension de navigateur représentant un bot d'automatisation avec statistiques.

Éléments visuels :
- Un robot stylisé (tête et bras) qui fait un geste de tap/click
- Un écran ou interface devant le robot montrant des statistiques (graphique en barres)
- Des lignes de connexion ou des ondes suggérant la communication/automatisation
- Palette : Orange corail (#FF6B6B) pour le robot, bleu clair (#4ECDC4) pour l'écran, blanc pour les détails

Style : Flat design moderne, icône vectorielle, formes géométriques simples
Fond : Transparent
Lisibilité : Optimisée pour 16x16 pixels minimum
```

## Prompt alternatif (style minimaliste)
```
Icône minimaliste pour extension d'automatisation.

Design ultra-simple :
- Un cercle orange (#FF6B6B) au centre
- À l'intérieur : un doigt stylisé pointant vers le bas (geste de tap)
- En bas du cercle : 3 petites barres horizontales de hauteurs différentes (graphique)
- Tout en blanc sur fond orange, ou orange sur fond transparent

Style : Flat design, formes géométriques pures, pas de dégradés
Contraste : Élevé pour visibilité maximale
```

## Instructions pour le générateur

1. **Générer d'abord la version 128x128** avec le prompt principal
2. **Redimensionner** pour créer les versions 48x48 et 16x16
3. **Ajuster les détails** : simplifier pour les petites tailles, enlever les éléments trop petits
4. **Vérifier la lisibilité** : l'icône doit être reconnaissable même à 16x16 pixels
5. **Exporter en PNG** avec fond transparent
6. **Nommer les fichiers** : `icon-16.png`, `icon-48.png`, `icon-128.png`

## Couleurs recommandées

- **Principal** : #FF6B6B (Orange corail - évoque Grindr)
- **Secondaire** : #FF5252 (Rouge plus foncé)
- **Accent** : #4ECDC4 (Bleu turquoise pour contraste)
- **Fond** : Transparent ou #FFFFFF (blanc)
- **Détails** : #FFFFFF (blanc) ou #2C3E50 (gris foncé)

- [ ] Style moderne et professionnel
